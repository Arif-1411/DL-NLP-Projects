# Deep Learning Projects

Deep learning projects built with TensorFlow and Keras. Covers Convolutional Neural Networks (CNN) for image classification and Recurrent Neural Networks (RNN, LSTM, Bidirectional LSTM) for sequential data and NLP tasks.

---

## Tools and Libraries

- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- NLTK
- Jupyter Notebook

---

## Topics Covered

**Convolutional Neural Networks (CNN)**
- Image classification
- Transfer learning (EfficientNet, VGG, ResNet)
- Data augmentation
- Custom CNN architecture

**Recurrent Neural Networks (RNN)**
- Simple RNN
- Long Short-Term Memory (LSTM)
- Bidirectional LSTM
- Sequence padding and tokenization

**Natural Language Processing (NLP)**
- Text preprocessing (tokenization, stemming, lemmatization)
- Stopword removal
- Word embeddings
- Sentiment analysis

**Model Techniques**
- Batch normalization and Dropout
- EarlyStopping and ReduceLROnPlateau
- Model checkpointing
- Training history visualization
- Confusion matrix and ROC curve

## How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/deep-learning.git
```

2. Install dependencies

```bash
pip install tensorflow nltk scikit-learn matplotlib seaborn pandas numpy wordcloud
```

3. Open any project notebook

```bash
jupyter notebook
```
